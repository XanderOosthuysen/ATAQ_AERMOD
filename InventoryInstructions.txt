================================================
ATAQ AERMOD - EMISSIONS INVENTORY INSTRUCTIONS
================================================

GENERAL RULES:
- source_id: Unique name for the source (No spaces, use underscores e.g., STACK_01).
- WKT (Well-Known Text): Defines the geometry in WGS84 (Longitude Latitude). 
  You can copy-paste WKT directly from QGIS or other GIS software.
- Pollutants (SO2, NO2, etc.): Leave as 0.0 if the source does not emit that pollutant.

--- 1. POINT SOURCES (point_sources.csv) ---
WKT Format     : POINT (Lon Lat)
elevation      : Base elevation above sea level (meters)
stack_height   : Release height above ground (meters)
stack_temp_k   : Exhaust gas temperature (Kelvin)  [Celsius + 273.15]
stack_velocity : Exhaust gas exit velocity (m/s)
stack_diameter : Inner diameter of the stack (meters)
Pollutants     : Emission rate in grams per second (g/s)

--- 2. AREA SOURCES (area_sources.csv) ---
WKT Format     : POLYGON ((Lon Lat, Lon Lat, ...))
elevation      : Base elevation above sea level (meters)
release_height : Release height above ground (meters)
szinit         : Initial vertical dispersion (meters). Usually 0.0 or release_height / 4.3.
Pollutants     : Emission rate in grams per second per square meter (g/s/m^2)

--- 3. LINE SOURCES (line_sources.csv) ---
WKT Format     : LINESTRING (Lon Lat, Lon Lat, ...)
elevation      : Base elevation above sea level (meters)
release_height : Release height above ground (meters)
width_m        : Width of the road/line (meters)
szinit         : Initial vertical dispersion (meters). Usually 0.0.
Pollutants     : Emission rate in grams per second per square meter (g/s/m^2)
